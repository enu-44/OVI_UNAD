¿Que es la programacion web?

La programación web o desarrollo web es una área muy grande en el mundo de la internet, el desarrollo web sorprende día a día con sus cambio evolutivos, no solo es interface sino también en sus productos, entre las necesidades de las páginas web también está la recolección de información, en la cual llegamos a la interconexión de las páginas con bases de datos, estos para ser organizados y administrados de una buena manera, como también aún podemos encontrar páginas web que no las requieren ya que son propias de mostrar información.

¿Que es php?

php es acrónimo de Hypertext Preprocessor es un lenguaje de código abierto que se incrusta en el html de la página y ayuda a simplificar ciertas acciones que son repetitivas con solo código, además de esto también nos ayuda con la conexión entre la página web y la base de datos, en cierto punto se vuelve el intermediario entre estas y nos ayuda también a mostrar la información que sea necesaria de mostrar en la pagina web.
